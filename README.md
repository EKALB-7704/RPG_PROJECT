*/ This is the read me for the rpg project
Area.cpp/area.h
Controls the area selection the character can go to there is a total of 4 choices the character can make forest desert mountains and swamp
Character_create.cpp/ Character_create.h
This allows for the player to choose their name age race and class. Name and age are used for their save file . Race determines their start location and stats determine their base stats

Encounters.cpp/encounters.h 
This determines when an enemy attacks the player and checks if it’s already been attacked in the area

Main_menu.cpp/main_menu.h 
This controls the main menu for the game where it has start game, load game and exit game

Store.cpp/store.h
These control the store in the game and allow the player to buy three items which are the sword which increases attack, the shield which increases defence and health which heals the player if they don’t have enough gold they are told to leave

Main.cpp 
This is the file that has all the functions and controls when they run which is inside the main loop
